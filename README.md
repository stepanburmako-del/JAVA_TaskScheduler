# Планировщик задач
## Описание
Консольное приложение на Java позволяющее распределить свои задачи по приоритету. Программа позволяет пользователю записать задачу, установить ее приоритет, а также поставить дату до какого числа данная задача должна быть выполнена.
## Особенности
* **Управление приоритетами:** Система поддерживает 5 уровней важности, что позволяет разделять задачи на критические и второстепенные.
* **Динамическая фильтрация:** Использование Java Stream API для мгновенного отображения задач, соответствующих заданному порогу приоритета.
* **Робастность (Отказоустойчивость):** Встроенная валидация защищает приложение от некорректного ввода (например, букв вместо цифр или неверных индексов при удалении) через обработку исключений `NumberFormatException` и `IndexOutOfBoundsException`.
* **Логирование:** Все ключевые события — запуск, добавление данных, успешное удаление и возникающие ошибки — фиксируются в журнале с помощью **Apache Log4j 2**.
* **Интерактивное меню:** Удобный цикл взаимодействия с пользователем через консольный интерфейс (CLI).
## Технологии
* **Язык:** Java (JDK 8+).
* **Библиотеки:** Log4j 2.25.3
* **Среда разработки:** IntelliJ IDEA.
## Структура кода
* `Task.java` — Модель данных (POJO), инкапсулирующая название, дату и приоритет задачи.
* `TaskScheduler.java` — Центральный узел приложения: содержит точку входа `main`, логику управления списком `ArrayList` и обработку меню.
* `log4j2.xml` — Конфигурационный файл, определяющий формат вывода логов в консоль и внешний файл `app.log`.
## Запуск
1. Убедитесь, что библиотеки `log4j-api` и `log4j-core` подключены к проекту через Maven или добавлены в **Project Structure -> Libraries**.
2. Поместите файл конфигурации `log4j2.xml` в папку ресурсов (`src/main/resources`).
3. Запустите класс `TaskScheduler`.
